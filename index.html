jquery.easie.js

Css3-like easing functions for jQuery

Features:

  • You may tweak the parameters of the easings in any way you like
  • Lookup tables make the implementation very fast!
  • Fully compatible with Css3 animation easings
  • Only around 0.6kB compressed and 1.2kB minified (jQuery.easing.js is around 5kB)
